# Хронология BuoyCalc Windows

Дата создания файла: 2026-06-30

Файл предназначен для последовательной фиксации изменений **только проекта BuoyCalc Windows**. Новые записи добавляются сверху или по версии, но без удаления старых записей.

## 2026-06-30 — исправление границы проекта

Версия / ветка / PR:

```text
project-governance-2026-06-30
```

Что изменено:

1. Уточнено, что `BuoyCalc Windows` и `PlataVM / плата питания` — абсолютно разные проекты.
2. Из архитектурной базы BuoyCalc Windows удалена фиксация PDU/PlataVM.
3. В управляющих документах оставлено только правило: не смешивать эти проекты и не переносить решения между ними без отдельной работы в соответствующем репозитории.

Зачем изменено:

Чтобы исключить ошибочное объединение расчётного приложения BuoyCalc Windows и аппаратного проекта платы питания PlataVM.

Затронутая архитектура BuoyCalc Windows:

Код и расчётная физика не затронуты. Исправлена только граница документации управления проектом.

Проверка:

1. Код не менялся.
2. Solver не менялся.
3. PDF не менялся.
4. Интерфейс не менялся.
5. PDU/PlataVM больше не описывается как часть архитектурной базы BuoyCalc Windows.

Что специально не изменялось:

1. Физика расчёта.
2. Старый solver.
3. Новый solver.
4. Gate.
5. Fallback-форма.
6. PDF-схема.
7. 2D-окно.
8. README версии `v0.46.4`.

Следующий шаг:

По текущему README следующий прикладной шаг — `v0.46.5`: перевести 2D-окно на пользовательскую терминологию без изменения физики solver.
