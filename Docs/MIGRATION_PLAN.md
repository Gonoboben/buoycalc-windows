# План переноса BuoyCalc на Windows

## Этап 1 — Windows v0.1

Цель: получить рабочее окно приложения и расчётное ядро.

Состав:

- C# модели;
- каталоги;
- расчёт;
- базовый интерфейс;
- текстовый отчёт.

## Этап 2 — Windows v0.2

Цель: улучшить редактор последовательности.

Добавить:

- нормальные выпадающие списки вместо ID;
- компактные карточки элементов;
- удаление конкретного элемента;
- перемещение вверх / вниз;
- дублирование элемента;
- суммарную длину линии;
- суммарную массу элементов.

## Этап 3 — Windows v0.3

Цель: сохранить проекты.

Добавить:

- JSON-хранение проектов;
- список проектов;
- создать / открыть / сохранить;
- дублировать проект;
- удалить проект.

## Этап 4 — Windows v0.4

Цель: библиотека элементов.

Перенести:

- пользовательские буи;
- пользовательские линии;
- пользовательские соединители;
- пользовательские якоря;
- источники данных.

## Этап 5 — Windows v0.5

Цель: отчёты.

Добавить:

- markdown-экспорт;
- PDF-экспорт;
- таблицы элементов;
- проверки;
- источники данных.

## Этап 6 — Windows v0.6

Цель: визуализация.

Добавить:

- цепочку постановки;
- 2D-схему сбоку;
- подписи глубины, сноса и длины линии.
