# BuoyCalc Windows v0.15

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.15 добавляет отдельное окно 2D-визуализации постановки. Теперь в правом блоке `Результат` есть кнопка `2D-схема`, которая открывает схематичный вид сбоку: поверхность воды, вертикальная цепочка элементов и дно.

## Что изменилось в v0.15

1. добавлено окно `Mooring2DWindow`;
2. добавлена кнопка `2D-схема` в блоке `Результат`;
3. 2D-схема открывается отдельно и не занимает место в главном окне;
4. схема использует текущую последовательность: буй → активные элементы → якорь;
5. отключённые элементы с выключенной галочкой `Учитывать в расчёте` в схему не попадают;
6. окно показывает поверхность воды, водную толщу и дно;
7. решение сделано без внешнего 3D-движка и без новых пакетов.

## Важное уточнение по визуализации

Текущая v0.15 — это первый стабильный 2D-экран. Он показывает порядок элементов, но пока не является масштабной инженерной графикой.

Дальше визуализация пойдёт отдельной веткой развития:

```text
v0.16 — масштабная 2D-схема по глубине, длине линии и сносу
v0.17 — 2D-схема в PDF-отчёте
v0.18 — простая 3D/isometric-схема без тяжёлого 3D-движка
v0.19 — улучшенная 3D-визуализация, если она реально нужна
```

## Как проверить

1. Запустить программу.
2. Проверить, что в правом блоке `Результат` появились две кнопки:
   - `2D-схема`;
   - `Экспорт PDF`.
3. Нажать `2D-схема`.
4. Проверить, что открылось отдельное окно `2D-схема постановки`.
5. Проверить, что в окне есть:
   - поверхность воды;
   - водная толща;
   - цепочка элементов;
   - дно / грунт.
6. Отключить один элемент галочкой `Учитывать в расчёте`.
7. Снова открыть `2D-схема`.
8. Проверить, что отключённый элемент не отображается.

## Что осталось от v0.14

PDF-экспорт остаётся активным. После расчёта можно нажать `Экспорт PDF`, выбрать путь сохранения и получить PDF-отчёт.
