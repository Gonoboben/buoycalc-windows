# BuoyCalc Windows v0.18.1

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.18.1 исправляет физически неверное отображение буя на 2D-схеме. Теперь буй рисуется на поверхности воды, а не под водой. Та же правка применена и к PDF-схеме.

## Что изменилось в v0.18.1

1. в `Mooring2DCanvas` буй поднят на линию поверхности воды;
2. линия постановки теперь начинается от нижней части буя и уходит к якорю;
3. узлы активных элементов распределяются вдоль линии от буя к якорю;
4. в `PdfReportBuilder` применена такая же логика для PDF-схемы;
5. дальнейший переход к 3D отложен;
6. следующий блок развития — физика расчёта, а не визуальные эффекты.

## Структура PDF после v0.18.1

```text
Страница 1: результат расчёта
Страница 2: 2D-схема постановки + текстовая цепочка
Страница 3+: таблица элементов + полный текстовый отчёт
```

## Как проверить

1. Запустить программу.
2. Нажать `2D-схема`.
3. Проверить, что буй находится на поверхности воды, а не под ней.
4. Выполнить расчёт через `Проверить схему и рассчитать`.
5. Нажать `Экспорт PDF`.
6. Открыть PDF.
7. На странице `Схема постановки` проверить, что буй тоже находится на поверхности воды.
