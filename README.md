# BuoyCalc Windows v0.5

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это не автоматическая конвертация SwiftUI-проекта, а портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.1 успешно запустилась на Windows в Visual Studio.

v0.2 улучшила общий вид главного окна.

## Как обновиться локально

Если репозиторий уже клонирован, в Visual Studio можно использовать:

```text
Git → Pull
```

Затем запустить проект заново.

## Как проверить библиотеку буёв

1. Открыть карточку `Буй`.
2. Выбрать встроенный пресет из выпадающего списка.
3. Нажать `Применить`.
4. Проверить, что объём, масса, площадь и Cd изменились.
5. Изменить название и параметры буя.
6. Нажать `Сохранить буй`.
7. Нажать `Обновить`.
8. Проверить, что пользовательский буй появился в списке.
9. Нажать `Рассчитать систему` и проверить отчёт.

## Как проверить сохранение проекта

1. Изменить название проекта.
2. Выбрать или сохранить буй.
3. Добавить или удалить элемент последовательности.
4. Нажать `Сохранить как...` и выбрать папку/имя файла.
5. Нажать `Новый`.
6. Нажать `Загрузить...` и выбрать сохранённый файл.
7. Проверить, что проект восстановился.

## Как запустить на Windows

### 1. Установить .NET SDK

Установить .NET SDK 8 или новее.

### 2. Открыть папку проекта

Открыть папку репозитория `buoycalc-windows` в Visual Studio, Rider или VS Code.

### 3. Восстановить пакеты

В терминале внутри папки проекта:

```bash
dotnet restore
```

### 4. Запустить

```bash
dotnet run
```
