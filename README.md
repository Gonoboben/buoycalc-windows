# BuoyCalc Windows v0.17.2

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.17.2 упрощает окно `2D-схема` до чистого viewer-окна. Теперь там нет верхнего тёмного колонтитула, нет блока инженерных параметров и нет кнопки `Закрыть`. Окно закрывается стандартным крестиком Windows.

## Что изменилось в v0.17.2

1. убран верхний тёмный колонтитул;
2. убран блок инженерных параметров;
3. убрана нижняя панель с кнопкой `Закрыть`;
4. окно стало ниже и компактнее;
5. внутри остались только:
   - заголовок `2D-схема постановки`;
   - бейдж версии;
   - графическое поле `Mooring2DCanvas`;
6. закрытие выполняется стандартным крестиком окна.

## Как проверить

1. Запустить программу.
2. Нажать `2D-схема`.
3. Проверить, что в окне нет:
   - тёмного верхнего колонтитула;
   - блока инженерных параметров;
   - кнопки `Закрыть`.
4. Проверить, что остался только графический viewer.
5. Закрыть окно стандартным крестиком сверху.
6. Нажать `Проверить схему и рассчитать` и убедиться, что текстовая цепочка всё ещё открывается отдельным окном подтверждения.

## Что осталось от v0.17

Геометрическая 2D-схема остаётся активной: поверхность воды, буй, линия, узлы активных элементов, якорь, дно, глубина и оценочный снос.

## Что осталось от v0.14

PDF-экспорт остаётся активным. После расчёта можно нажать `Экспорт PDF`, выбрать путь сохранения и получить PDF-отчёт.
