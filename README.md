# BuoyCalc Windows v0.17.1

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.17.1 разделяет визуализацию и текстовую проверку цепочки по двум разным окнам. Окно `2D-схема` теперь содержит только графику и инженерные параметры, а текстовая проверка последовательности остаётся в отдельном окне перед расчётом.

## Что изменилось в v0.17.1

1. из окна `2D-схема` убран нижний текстовый список цепочки;
2. графическая схема больше не должна залезать на блоки ниже;
3. окно `2D-схема` стало отдельным окном только для графики;
4. окно проверки перед расчётом остаётся отдельным окном только для текстовой цепочки;
5. инженерные подписи по глубине, длине линии, сносу и `L/Depth` остались в верхней части 2D-окна;
6. графический контрол `Mooring2DCanvas` остался без изменения логики.

## Как проверить

1. Запустить программу.
2. Нажать `2D-схема`.
3. Проверить, что в окне есть только:
   - инженерные параметры;
   - графическое поле;
   - кнопка закрытия.
4. Проверить, что текстовый список цепочки больше не находится под графиком.
5. Нажать `Проверить схему и рассчитать`.
6. Проверить, что текстовая цепочка открывается отдельным окном подтверждения.
7. Выполнить расчёт.
8. Снова открыть `2D-схема` и проверить, что снос влияет на наклон линии.

## Что осталось от v0.17

Геометрическая 2D-схема остаётся активной: поверхность воды, буй, линия, узлы активных элементов, якорь, дно, глубина и оценочный снос.

## Что осталось от v0.14

PDF-экспорт остаётся активным. После расчёта можно нажать `Экспорт PDF`, выбрать путь сохранения и получить PDF-отчёт.
