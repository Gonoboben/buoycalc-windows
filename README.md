# BuoyCalc Windows v0.19

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.19 добавляет первый инженерный вход для будущей матмодели MD&D-типа: профиль течения по глубине. Теперь вместо одного значения `CurrentSpeed` можно задать таблицу слоёв с компонентами течения `U / V / W` и плотностью воды.

## Что изменилось в v0.19

1. добавлена модель точки профиля течения;
2. добавлено окно `Профиль течения по глубине`;
3. в профиль входят поля:
   - глубина, м;
   - U East, м/с;
   - V North, м/с;
   - W Vertical, м/с;
   - плотность воды, кг/м³;
4. профиль можно включить или отключить;
5. профиль сохраняется в JSON проекта;
6. профиль восстанавливается при загрузке проекта;
7. отчёт показывает таблицу профиля течения;
8. расчёт пока использует профиль как переходную оценку: берётся максимальная горизонтальная скорость `max |Uгор|`;
9. старый режим с одним `CurrentSpeed` остаётся рабочим.

## Почему это важно

Дальше расчёт должен идти не от условной картинки, а от математической модели:

```text
профиль течения по глубине
↓
разбиение линии на сегменты
↓
локальная сила сопротивления на каждом сегменте
↓
натяжения и равновесие
↓
координаты X/Y/Z
↓
2D/PDF как отображение рассчитанного состояния
```

v0.19 — это подготовка входных данных для такой модели.

## Ограничение v0.19

В этой версии ещё нет полноценной послойной интеграции сил по сегментам. Профиль уже вводится, сохраняется и попадает в отчёт, но расчёт временно использует его как эффективную скорость:

```text
U_eff = max sqrt(U^2 + V^2)
```

Послойный расчёт по глубине будет следующим этапом.

## Как проверить

1. Запустить программу.
2. Нажать `Профиль течения...` в верхней панели или в блоке условий.
3. Включить галочку `Использовать профиль течения`.
4. Отредактировать несколько точек профиля.
5. Закрыть окно крестиком.
6. Выполнить расчёт через `Проверить схему и рассчитать`.
7. Проверить в блоке `Результат`, что появилась строка `Течение расчётное`.
8. Проверить в отчёте раздел `Профиль течения по глубине`.
9. Сохранить проект.
10. Загрузить проект и проверить, что профиль восстановился.
