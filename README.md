# BuoyCalc Windows v0.21.3

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.21.3 — техническая чистка после добавления полного текстового отчёта. В `MainWindow.axaml` устаревшие свойства `TextBox.Watermark` заменены на `TextBox.PlaceholderText`.

## Что изменилось в v0.21.3

1. убраны предупреждения Avalonia `AVLN5001: TextBox.Watermark is obsolete` из главного окна;
2. `Watermark="..."` заменён на `PlaceholderText="..."`;
3. главное окно отформатировано более читаемо;
4. версия в заголовке главного окна обновлена до `v0.21.3`.

## Что было добавлено в v0.21.1–v0.21.2

1. PDF больше не обрезает полный текстовый отчёт;
2. строка `... отчёт обрезан для компактного PDF` больше не должна появляться;
3. полный текстовый отчёт продолжается на следующих страницах PDF;
4. добавлено окно `ReportTextWindow`;
5. в верхнюю панель главного окна добавлена кнопка `Отчёт текстом...`;
6. окно отчёта показывает полный `ReportText` без обрезки;
7. исправлена совместимость скролла окна отчёта с текущей версией Avalonia.

## Что осталось от v0.21

v0.21 добавляет первый расчёт натяжений по сегментам линии. После v0.20 у нас уже были сегменты, глубины и локальные силы течения. Теперь для каждого сегмента оцениваются вес в воде, накопленная горизонтальная сила, накопленная вертикальная сила, натяжение и угол от вертикали.

## Что важно

Это ещё не полный `moordyn`. Сейчас натяжение считается по накопленным силам снизу вверх, но координаты линии пока не итерационные. То есть это переходная физическая модель:

```text
сегменты линии
↓
локальные силы
↓
накопленные силы снизу вверх
↓
оценочное натяжение
↓
следующий этап: координаты узлов и форма линии
```

## Как проверить

1. Запустить сборку.
2. Проверить, что предупреждения `TextBox.Watermark is obsolete` больше не появляются для `MainWindow.axaml`.
3. Запустить программу.
4. Выполнить расчёт.
5. Нажать `Отчёт текстом...`.
6. Проверить экспорт PDF.
