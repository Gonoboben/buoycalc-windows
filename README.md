# BuoyCalc Windows v0.12

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.12 добавляет выбор грунта и делает расчёт удержания якоря более прозрачным. Теперь грунт выбирается в главном окне, сохраняется в JSON проекта и участвует в расчёте запаса якоря.

## Что изменилось в v0.12

1. в главном окне в блоке `Условия постановки` появился выбор грунта;
2. доступны грунты из `SeabedCatalog`: неизвестный грунт, ил, песок, камень;
3. выбранный грунт сохраняется в JSON проекта;
4. выбранный грунт восстанавливается при загрузке проекта;
5. запас якоря теперь считается с учётом грунтового множителя;
6. расчёт якоря стал прозрачнее: `вес в воде × K якоря × K типа × K грунта`;
7. в отчёте добавлены коэффициенты якоря, типа якоря и грунта;
8. в проверках появляется предупреждение, если грунт оставлен неизвестным;
9. строка якоря в таблице показывает тип якоря и грунтовый множитель.

## Формула удержания якоря

```text
Удержание якоря, кг = вес якоря в воде × K якоря × K типа × K грунта
```

Где:

```text
K якоря  — коэффициент из карточки якоря
K типа   — дополнительная поправка по типу якоря
K грунта — множитель выбранного грунта
```

Текущие учебные множители типа якоря:

```text
Deadweight / бетонный груз: 1.00
Mushroom / грибовидный:     1.15
Plate / плитный:            1.25
```

## Как проверить

1. Запустить программу.
2. В блоке `Условия постановки` выбрать грунт, например `Ил`.
3. Нажать `Рассчитать систему`.
4. В блоке `Результат` проверить строку `Грунт`.
5. В таблице элементов проверить строку `Якорь`.
6. В отчёте найти раздел `Якорь` и формулу удержания.
7. Поменять грунт на `Камень` и снова нажать расчёт.
8. Проверить, что запас якоря изменился.
