# BuoyCalc Windows v0.22

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.22 добавляет предварительные расчётные узлы линии `X/Z`. После v0.20–v0.21 у нас уже есть сегменты, локальные силы течения, вес в воде и оценочные натяжения. Теперь по углам сегментов строится набор узлов, который станет основой для будущей 2D-схемы по рассчитанной форме.

## Что изменилось в v0.22

1. добавлен сервис `MooringNodeAnalyzer`;
2. добавлена модель `MooringNodeRow`;
3. расчёт строит узлы сверху вниз:
   - верхний узел линии;
   - узел после каждого сегмента;
   - накопленная длина вдоль линии `s`;
   - предварительная координата `X`;
   - предварительная глубина `Z`;
   - длина сегмента;
   - угол сегмента;
   - натяжение сегмента;
4. в разделе `Итоги` появился предварительный горизонтальный снос по узлам `X/Z`;
5. в отчёт добавлен раздел `Расчётные узлы линии X/Z`.

## Что важно

Это ещё не финальный `moordyn` и не полная равновесная форма. Узлы строятся по текущим оценочным углам сегментов:

```text
ΔX = L × sin(угол)
ΔZ = L × cos(угол)
```

Затем глубина нормируется по оценочной глубине сегментов. Это переходная модель, нужная для следующего этапа — итерационного равновесного расчёта и замены 2D-схемы.

## Как проверить

1. Запустить программу.
2. Выполнить расчёт.
3. Нажать `Отчёт текстом...`.
4. Найти раздел `Расчётные узлы линии X/Z`.
5. Проверить, что в таблице есть:
   - номер узла;
   - номер сегмента;
   - элемент;
   - s, м;
   - X, м;
   - Z, м;
   - угол;
   - T, кН.
6. В разделе `Итоги` проверить строку `Предварительный горизонтальный снос по узлам X/Z`.
