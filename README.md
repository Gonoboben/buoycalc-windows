# BuoyCalc Windows v0.13

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.13 добавляет визуальную схему постановки в интерфейс. Теперь в центральной колонке видно не только редактор карточек, но и автоматическую вертикальную цепочку: буй → активные элементы последовательности → якорь.

## Что изменилось в v0.13

1. в `MainWindowViewModel` добавлена коллекция `SequenceDiagramLines`;
2. схема строится автоматически из выбранного буя, активных элементов и выбранного якоря;
3. схема обновляется при изменении последовательности;
4. схема обновляется при смене буя;
5. схема обновляется при смене якоря;
6. в главном окне появился блок `Визуальная схема`;
7. схема показывает порядок сверху вниз;
8. неактивные элементы в схему не попадают;
9. решение сделано без внешних пакетов и без сложной графики, чтобы не ломать сборку.

## Как выглядит логика схемы

```text
● Буй
↓
○ Соединитель
↓
○ Линия
↓
○ Прибор / нагрузка
↓
■ Якорь
```

## Как проверить

1. Запустить программу.
2. В центральной колонке найти блок `Визуальная схема`.
3. Проверить, что первая строка — выбранный буй.
4. Проверить, что последняя строка — выбранный якорь.
5. Нажать `+ линия`, `+ соединитель` или `+ прибор`.
6. Проверить, что новый элемент появился в схеме.
7. Снять галочку `Учитывать в расчёте` у элемента.
8. Проверить, что этот элемент исчез из схемы.
9. Переместить элемент вверх или вниз.
10. Проверить, что порядок в схеме изменился.

## Что осталось от v0.12

Выбор грунта и расчёт якоря по формуле остаются активными:

```text

```
