# BuoyCalc Windows v0.8

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это не автоматическая конвертация SwiftUI-проекта, а портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Архитектурное решение

Главное окно постепенно становится окном проекта:

```text
Главное окно проекта
├─ выбрать буй из библиотеки
├─ собрать последовательность элементов
├─ задать условия
├─ выбрать / задать якорь
└─ рассчитать / сформировать отчёт
```

Отдельное окно библиотеки постепенно становится общим редактором элементов:

```text
Библиотека элементов
├─ буи
├─ линии
├─ соединители
├─ приборы
└─ якоря
```

## Как проверить библиотеку соединителей

1. В верхней панели главного окна нажать `Библиотека элементов...`.
2. Открыть вкладку `Соединители`.
3. Выбрать встроенный соединитель и посмотреть параметры.
4. Нажать `Новый соединитель`.
5. Заполнить название, тип, массу, объём, MBL, площадь и Cd.
6. Нажать `Сохранить`.
7. Проверить, что новый соединитель появился в списке.
8. Закрыть окно библиотеки.
9. В главном окне добавить элемент `+ соединитель`.
10. Открыть выпадающий список `Пресет соединителя` и выбрать пользовательский соединитель.
11. Нажать `Рассчитать систему`.

## Как проверить сохранение проекта

1. Изменить название проекта.
2. Создать пользовательский соединитель.
3. Добавить его в последовательность.
4. Нажать `Сохранить как...` и выбрать папку/имя файла.
5. Нажать `Новый`.
6. Нажать `Загрузить...` и выбрать сохранённый файл.
7. Проверить, что проект восстановился.

## Как запустить на Windows

### 1. Установить .NET SDK

Установить .NET SDK 8 или новее.

### 2. Открыть папку проекта

Открыть папку репозитория `buoycalc-windows` в Visual Studio, Rider или VS Code.

### 3. Восстановить пакеты

В терминале внутри папки проекта:

```bash
dotnet restore
```

### 4. Запустить

```bash
dotnet run
```
