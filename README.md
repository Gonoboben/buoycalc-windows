# BuoyCalc Windows v0.46.1

Windows-ветка проекта BuoyCalc на C# + Avalonia.

## Статус

v0.46.1 — финальная чистка PDF-отчёта перед тестированием. Расчётная физика, solver, 2D-координаты, PDF-схема и release build контур не изменены. Добавлен фильтр полного текстового отчёта внутри PDF, чтобы убрать устаревшие методические строки про старую основную/fallback форму и ранние версии solver-слоя.

## Что добавлено

1. Новый сервис:

```text
Services/PdfReportTextCleanup.cs
```

2. Он применяется в:

```text
Services/PdfReportStructureGuide.cs
```

3. Фильтр убирает из PDF-потока:

```text
блок Выбор основной формы
primaryShape...
MooringPrimaryShapeGate / Selector
строки про fallback как объект PDF-отчёта
устаревшие строки v0.39/v0.40 о том, что 2D/PDF ещё не заменены
```

4. Фильтр также переименовывает пользовательские формулировки:

```text
альтернативная форма -> форма с дискретными элементами
старая оценка -> исходная оценка
Top T старая -> Top T исходная
```

5. Версия приложения:

```text
v0.46.1 - PDF report cleanup
```

## Что не изменилось

1. Физика расчёта не изменилась.
2. Solver не изменился.
3. Gate не удалён.
4. Fallback-форма не удалена из приложения.
5. PDF-схема по-прежнему берёт координаты из расчётного слоя дискретных нагрузок.
6. Release build workflow остаётся тем же.

## Как проверить

1. Выполнить расчёт.
2. Экспортировать PDF.
3. Проверить, что в PDF нет:

```text
Выбор основной формы
primaryShape
основная форма X/Z
MooringShapeSolver fallback
v0.39 добавляет диагностический
```

4. Проверить, что в PDF есть:

```text
Структура PDF-отчёта v0.46.1
форма с дискретными элементами
```

5. Проверить CI:

```text
BuoyCalc Windows Build: success
```
