# BuoyCalc Windows v0.18

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.18 добавляет геометрическую 2D-схему в PDF-отчёт. Теперь при экспорте PDF создаётся отдельная страница `Схема постановки`, где рисуются поверхность воды, буй, линия, активные элементы, якорь, дно и оценочный снос.

## Что изменилось в v0.18

1. `PdfReportBuilder` теперь получает числовые параметры 2D-визуализации:
   - глубину;
   - длину линии;
   - оценочный снос;
2. в PDF добавлена отдельная страница `Схема постановки`;
3. на странице схемы рисуются:
   - поверхность воды;
   - водная толща;
   - буй;
   - линия постановки;
   - узлы активных элементов;
   - якорь;
   - дно / грунт;
   - подпись сноса;
4. текстовая цепочка элементов остаётся под графикой как проверочный список;
5. таблица элементов и полный текстовый отчёт перенесены на следующую страницу;
6. экспорт PDF по-прежнему запускается только после выполненного расчёта.

## Структура PDF после v0.18

```text
Страница 1: результат расчёта
Страница 2: 2D-схема постановки + текстовая цепочка
Страница 3+: таблица элементов + полный текстовый отчёт
```

## Как проверить

1. Запустить программу.
2. Выполнить расчёт через `Проверить схему и рассчитать`.
3. Нажать `Экспорт PDF`.
4. Сохранить PDF.
5. Открыть PDF.
6. Проверить, что появилась отдельная страница `Схема постановки`.
7. Проверить, что на странице есть графика:
   - поверхность воды;
   - буй;
   - линия;
   - узлы элементов;
   - якорь;
   - дно;
   - подпись сноса.

## Что осталось от v0.17.2

Окно `2D-схема` остаётся чистым viewer-окном: без верхнего тёмного колонтитула, без блока параметров и без кнопки закрытия.
