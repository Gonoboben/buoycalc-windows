# BuoyCalc Windows v0.11.4

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.11.4 исправляет состав и читаемость таблицы элементов. Теперь таблица показывает всю постановку: буй, элементы последовательности и якорь. Шрифт таблицы уменьшен, строки стали компактнее.

## Что изменилось в v0.11.4

1. в таблицу элементов добавлена строка `Буй`;
2. в таблицу элементов добавлена строка `Якорь`;
3. элементы последовательности теперь находятся между буем и якорем;
4. строка буя показывает чистый вклад плавучести в колонке `Вес в воде`;
5. строка якоря показывает вес якоря в воде и запас якоря;
6. шрифт таблицы уменьшен;
7. заголовок окна обновлён до `v0.11.4`.


## Как проверить

1. Запустить программу.
2. Нажать `Рассчитать систему`.
3. В правой колонке открыть `Таблица элементов`.
4. Проверить, что первая строка — `Буй`.
5. Проверить, что последняя строка — `Якорь`.
6. Проверить, что между ними идут соединители, линии и приборы.
7. Проверить, что шрифт стал меньше.
