# BuoyCalc Windows v0.14

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.14 добавляет первый PDF-экспорт из приложения. После расчёта можно нажать `Экспорт PDF`, выбрать путь сохранения и получить PDF-отчёт со сводкой, схемой, таблицей элементов и полным текстовым отчётом.

## Что изменилось в v0.14

1. добавлен сервис `PdfReportBuilder`;
2. PDF строится из уже рассчитанных данных интерфейса;
3. добавлена кнопка `Экспорт PDF` в правом блоке `Результат`;
4. если расчёт ещё не выполнен, экспорт не запускается;
5. PDF содержит:
   - название проекта;
   - краткий результат;
   - визуальную схему постановки;
   - таблицу элементов;
   - полный текстовый отчёт;
6. файл сохраняется через стандартный диалог Windows;
7. после сохранения путь к PDF показывается в статусе проекта.

## Как проверить

1. Запустить программу.
2. Нажать `Проверить схему и рассчитать`.
3. В окне схемы нажать `Всё верно, рассчитать`.
4. Проверить, что появились результат, таблица и отчёт.
5. Нажать `Экспорт PDF`.
6. Выбрать папку и имя файла.
7. Проверить, что PDF-файл появился по выбранному пути.
8. Открыть PDF и проверить разделы:
   - результат;
   - визуальная схема постановки;
   - таблица элементов;
   - полный текстовый отчёт.

## Техническое примечание

PDF-экспорт сделан отдельным сервисом и не меняет расчётное ядро. В этой версии PDF является первым рабочим вариантом: формат компактный, без сложной вёрстки. Дальше его можно улучшать визуально и добавлять графическую схему.

## Что осталось от v0.13.1

Перед расчётом всё ещё открывается отдельное окно проверки схемы. Расчёт выполняется только после подтверждения пользователя.
