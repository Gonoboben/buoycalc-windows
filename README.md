# BuoyCalc Windows v0.16

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.16 развивает 2D-визуализацию: окно `2D-схема` теперь показывает не только порядок элементов, но и инженерные масштабные параметры — глубину, суммарную длину линии, оценочный снос и отношение `L/Depth`.

## Что изменилось в v0.16

1. в `MainWindowViewModel` добавлены параметры 2D-визуализации;
2. 2D-окно показывает `Глубина`;
3. 2D-окно показывает `Длина линии`;
4. после расчёта 2D-окно показывает `Оценочный снос`;
5. 2D-окно показывает отношение `L/Depth`;
6. добавлен статус:
   - `OK: длина линии не меньше глубины`;
   - `WARNING: линия короче глубины`;
7. при изменении последовательности параметры 2D-схемы обновляются;
8. при расчёте снос обновляется по результату расчёта.

## Важное уточнение по визуализации

Текущая v0.16 — это уже инженерно-информативная 2D-схема, но геометрическая линия пока ещё не прорисовывается как настоящая кривая/ломаная по масштабу.

Дальше визуализация идёт отдельной веткой:

```text
v0.17 — геометрическая 2D-схема: глубина, линия, снос, якорь
v0.18 — 2D-схема в PDF-отчёте
v0.19 — простая 3D/isometric-схема без тяжёлого 3D-движка
v0.20 — улучшенная 3D-визуализация, если она реально нужна
```

## Как проверить

1. Запустить программу.
2. Нажать `2D-схема` до расчёта.
3. Проверить, что отображаются:
   - глубина;
   - длина линии;
   - L/Depth;
   - статус по длине линии.
4. Нажать `Проверить схему и рассчитать`.
5. Подтвердить схему и выполнить расчёт.
6. Снова открыть `2D-схема`.
7. Проверить, что появился `Оценочный снос`.
8. Изменить длину линии или отключить участок линии.
9. Проверить, что `Длина линии`, `L/Depth` и статус обновились.

## Что осталось от v0.15

2D-схема всё ещё открывается отдельным окном через кнопку `2D-схема`. Отключённые элементы не попадают в цепочку.

## Что осталось от v0.14

PDF-экспорт остаётся активным. После расчёта можно нажать `Экспорт PDF`, выбрать путь сохранения и получить PDF-отчёт.
