# BuoyCalc Windows v0.4.1

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это не автоматическая конвертация SwiftUI-проекта, а портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.1 успешно запустилась на Windows в Visual Studio.

v0.2 улучшила общий вид главного окна.

## Что изменилось в v0.4.1

1. добавлены маленькие `?` рядом с ключевыми исходными данными в левой колонке;
2. при наведении на `?` показывается подсказка, что означает параметр и где брать значение;
3. добавлен видимый путь файла проекта;
4. добавлена кнопка `Сохранить как...` с системным диалогом выбора пути;
5. кнопка `Загрузить...` открывает системный диалог выбора JSON-файла;
6. соединитель теперь считается как один элемент последовательности;
7. поле `Количество` у соединителей убрано из интерфейса;
8. если нужно несколько соединителей, нужно добавить или продублировать несколько элементов.

## Как обновиться локально

Если репозиторий уже клонирован, в Visual Studio можно использовать:

```text
Git → Pull
```

Или через терминал, если установлен Git:

```bash
git pull
```

Затем запустить проект заново.

## Как проверить сохранение

1. Изменить название проекта.
2. Добавить или удалить элемент последовательности.
3. Нажать `Сохранить как...` и выбрать папку/имя файла.
4. Нажать `Новый`.
5. Нажать `Загрузить...` и выбрать сохранённый файл.
6. Проверить, что проект восстановился.
7. Нажать `Рассчитать систему`.

## Как запустить на Windows

### 1. Установить .NET SDK

Установить .NET SDK 8 или новее.

### 2. Открыть папку проекта

Открыть папку репозитория `buoycalc-windows` в Visual Studio, Rider или VS Code.

### 3. Восстановить пакеты

В терминале внутри папки проекта:

```bash
dotnet restore
```

### 4. Запустить

```bash
dotnet run
```
