# BuoyCalc Windows v0.22.2

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.22.2 уточняет смысл нижнего граничного узла: это не точка дна, а точка крепления линии к якорю. Если якорь имеет высоту около 0,5 м, то точка крепления линии должна быть примерно на 0,5 м выше дна.

## Что изменилось в v0.22.2

1. последний узел теперь подписан как `Якорь / точка крепления линии`;
2. у якорного узла длина сегмента `Lсег` равна 0, потому что это граничный узел, а не сам сегмент;
3. глубина якорного узла снова трактуется как глубина точки крепления линии;
4. в разделе `Итоги` добавлена строка `Глубина точки крепления линии к якорю`;
5. в отчёт добавлено пояснение: глубина якорного узла не обязана быть равна глубине дна.

## Что было добавлено в v0.22–v0.22.1

1. добавлен сервис `MooringNodeAnalyzer`;
2. добавлена модель `MooringNodeRow`;
3. расчёт строит узлы сверху вниз:
   - верхний узел линии;
   - узел после каждого сегмента;
   - накопленная длина вдоль линии `s`;
   - предварительная координата `X`;
   - предварительная глубина `Z`;
   - длина сегмента;
   - угол сегмента;
   - натяжение сегмента;
4. в разделе `Итоги` появился предварительный горизонтальный снос по узлам `X/Z`;
5. в отчёт добавлен раздел `Расчётные узлы линии X/Z`;
6. верхний узел явно подписан как `Буй / верхний конец линии`.

## Что важно

Это ещё не финальный `moordyn` и не полная равновесная форма. Узлы строятся по текущим оценочным углам сегментов:

```text
ΔX = L × sin(угол)
ΔZ = L × cos(угол)
```

Затем глубина нормируется до точки крепления линии к якорю, а не обязательно до дна.

## Логика расчёта сейчас

```text
профиль течения
↓
сегменты линии
↓
локальные силы
↓
натяжения и углы
↓
узлы X/Z: буй → линия → точка крепления к якорю
↓
следующий этап: 2D-схема по рассчитанным узлам
```

## Как проверить

1. Запустить программу.
2. Выполнить расчёт.
3. Нажать `Отчёт текстом...`.
4. Найти раздел `Расчётные узлы линии X/Z`.
5. Проверить, что первый узел подписан как `Буй / верхний конец линии`.
6. Проверить, что последний узел подписан как `Якорь / точка крепления линии`.
7. Проверить, что у якорного узла `Lсег = 0`.
8. Проверить, что глубина якорного узла соответствует точке крепления линии, а не обязательно полной глубине дна.
