# BuoyCalc Windows v0.11.6

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.11.6 приводит числовые значения в таблице элементов к единому формату: до 4 знаков после запятой. Расчёт внутри остаётся в `double`, а в интерфейс передаются уже отформатированные строки.

## Что изменилось в v0.11.6

1. добавлена модель `ElementCalculationDisplayRow`;
2. таблица элементов теперь показывает числа в формате `0.####`;
3. значения не должны выводиться длинными дробями;
4. Markdown-таблица в отчёте тоже переведена на формат до 4 знаков после запятой;
5. расчётное ядро не упрощалось: округление применяется только к отображению и отчёту.

## Как проверить

1. Запустить программу.
2. Нажать `Рассчитать систему`.
3. В правой колонке открыть `Таблица элементов`.
4. Проверить, что значения отображаются максимум с 4 знаками после запятой.
5. Проверить, что первая строка — `Буй`.
6. Проверить, что последняя строка — `Якорь`.
7. Проверить раздел `Таблица элементов` в Markdown-отчёте.
