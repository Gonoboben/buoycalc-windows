# BuoyCalc Windows v0.46.4

Windows-ветка проекта BuoyCalc на C# + Avalonia.

## Статус

v0.46.4 — продолжение приведения проекта к рабочему состоянию после контрольной отметки. Физика расчёта не менялась. Сделаны пользовательские термины для главного окна и PDF-схемы.

## Что изменено

1. Верхняя кнопка главного окна безопасно переименовывается через code-behind:

```text
Отчёт текстом... -> Полный отчёт...
```

2. Верхний бейдж версии главного окна теперь берётся из:

```text
AppInfo.DisplayVersion
```

3. Обновлён файл:

```text
Views/MainWindow.axaml.cs
```

4. В пользовательском PDF убран необъяснённый технический статус:

```text
форма: WARNING
```

5. Вместо него выводится пользовательская формулировка:

```text
форма: требует проверки
```

6. Для сошедшейся формы выводится:

```text
форма: ОК
```

7. Обновлён файл:

```text
Services/PdfReportBuilder.cs
```

8. Версия приложения:

```text
v0.46.4 - пользовательские статусы PDF
```

## Что пока не изменено

1. 2D-окно ещё требует отдельной аккуратной правки: сейчас там исторически смешаны исходная и форма с дискретными элементами.
2. 3D-окно пока не добавлялось.
3. Физика solver пока не усиливалась.
4. Warning policy пока полностью применена только к пользовательскому PDF.

## Что не изменилось

1. Физика расчёта не изменилась.
2. Solver не изменился.
3. Gate не удалён.
4. Fallback-форма не удалена из приложения.
5. PDF-схема берёт координаты из расчётного слоя дискретных нагрузок.
6. Release build workflow остаётся тем же.

## Следующий шаг

v0.46.5:

```text
1. Перевести 2D-окно на ту же пользовательскую терминологию.
2. В 2D показывать форму с дискретными элементами как выбранную форму, без старого сравнения основной/альтернативной формы.
3. Подготовить основу 3D-окна без изменения solver.
```

## Как проверить

1. Запустить приложение.
2. Проверить верхнюю кнопку:

```text
Полный отчёт...
```

3. Выполнить расчёт.
4. Экспортировать PDF.
5. Проверить, что на схеме PDF нет:

```text
форма: WARNING
```

6. Проверить, что вместо этого есть:

```text
форма: требует проверки
```

7. Проверить CI:

```text
BuoyCalc Windows Build: success
```
