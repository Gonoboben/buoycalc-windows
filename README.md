# BuoyCalc Windows v0.9

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.9 добавляет библиотеку якорей и выбор якоря из библиотеки в главном окне. Расчётная формула удержания пока осталась прежней, но теперь она получает данные из выбранного якоря.

## Что уже есть

1. условия среды;
2. выбор буя из библиотеки;
3. последовательность постановки;
4. линии из библиотеки;
5. соединители из библиотеки;
6. приборы / нагрузки пока вручную;
7. выбор якоря из библиотеки;
8. расчёт плавучести;
9. расчёт сопротивления течению;
10. упрощённая волновая нагрузка;
11. натяжение;
12. удержание якоря;
13. проверки системы;
14. текстовый отчёт;
15. сохранение проекта в JSON;
16. загрузка проекта из JSON;
17. отдельное окно `Библиотека элементов`.

## Что изменилось в v0.9

1. добавлена модель `AnchorLibraryItem`;
2. добавлен сервис `AnchorLibraryStorage`;
3. в окне `Библиотека элементов` появилась вкладка `Якоря`;
4. можно создать пользовательский якорь;
5. можно сохранить якорь;
6. можно удалить пользовательский якорь;
7. встроенные якоря удалить нельзя;
8. пользовательские якоря сохраняются в JSON;
9. в главном окне якорь выбирается из списка;
10. поля якоря в главном окне стали read-only;
11. в проекте сохраняется выбранный ID якоря.

## Как проверить v0.9

1. В верхней панели главного окна нажать `Библиотека элементов...`.
2. Открыть вкладку `Якоря`.
3. Выбрать встроенный якорь и посмотреть параметры.
4. Нажать `Новый якорь`.
5. Заполнить название, тип, материал, массу, объём и коэффициент удержания.
6. Нажать `Сохранить`.
7. Закрыть окно библиотеки.
8. В главном окне в карточке `Якорь и запас` выбрать созданный якорь.
9. Нажать `Рассчитать систему`.
10. Проверить, что отчёт использует выбранный якорь.

## Когда улучшаем расчёт

Следующий крупный расчётный этап — **v0.11**.

Перед ним стоит добавить библиотеку приборов / нагрузок в **v0.10**, чтобы расчёт опирался не на разрозненные ручные поля, а на нормальные паспортные элементы.

В v0.11 план расчёта:

1. проверка слабого звена по всей цепочке;
2. запас по MBL для линии и каждого соединителя;
3. вес в воде по каждому элементу;
4. отдельный расчёт якоря по типу;
5. поправка удержания на грунт;
6. более понятные предупреждения в отчёте;
7. таблица вкладов элементов в вес, сопротивление и запас.

## Следующий шаг

Рекомендуемый v0.10:

1. добавить библиотеку приборов / нагрузок;
2. заменить ручной ввод прибора в последовательности на выбор из библиотеки;
3. после этого перейти к улучшенному расчётному ядру v0.11.
