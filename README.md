# BuoyCalc Windows v0.1

Стартовая Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это не автоматическая конвертация SwiftUI-проекта, а первый шаг портирования:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Что уже перенесено

В v0.1 перенесено расчётное ядро и базовый desktop-интерфейс:

1. условия среды;
2. буй;
3. последовательность постановки;
4. линии;
5. соединители;
6. приборы / нагрузки;
7. якорь;
8. грунт;
9. расчёт плавучести;
10. расчёт сопротивления течению;
11. упрощённая волновая нагрузка;
12. натяжение;
13. удержание якоря;
14. проверки системы;
15. текстовый отчёт.

## Что пока не перенесено

Пока не перенесены:

1. сохранение проектов;
2. пользовательская библиотека элементов;
3. PDF-экспорт;
4. 2D-визуализация;
5. визуальная цепочка постановки;
6. источники данных;
7. импорт / экспорт;

Эти блоки лучше переносить после того, как v0.1 стабильно запускается на Windows.

## Как запустить на Windows

### 1. Установить .NET SDK

Установить .NET SDK 8 или новее.

### 2. Открыть папку проекта

Открыть папку `BuoyCalc_Windows_v0_1` в Visual Studio, Rider или VS Code.

### 3. Восстановить пакеты

В терминале внутри папки проекта:

```bash
dotnet restore
```

### 4. Запустить

```bash
dotnet run
```

## Технологии

- C#
- .NET 8
- Avalonia 12.0.5
- XAML
- MVVM-подход без внешних MVVM-библиотек

## Важное ограничение

Этот архив был подготовлен как стартовая структура. В текущей среде не установлен `dotnet SDK`, поэтому проект не был скомпилирован здесь. Первую компиляцию нужно выполнить на Windows.

## Что делать дальше

Рекомендуемый порядок:

1. открыть проект в Visual Studio / Rider;
2. выполнить `dotnet restore`;
3. выполнить `dotnet run`;
4. исправить возможные мелкие ошибки XAML / пакетов;
5. после запуска перенести сохранение проектов;
6. затем перенести библиотеку элементов;
7. затем перенести PDF и визуализацию.
