# BuoyCalc Windows v0.6.1

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это не автоматическая конвертация SwiftUI-проекта, а портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

2. из карточки `Буй` убрана кнопка открытия библиотеки;

4. в окне библиотеки добавлены подсказки `?` к параметрам буя:
   - название;
   - объём;
   - масса;
   - проецируемая площадь;
   - Cd;
   - заметка / источник;
5. верхняя панель библиотеки подготовлена под дальнейшее расширение разделов.

Пользовательская библиотека буёв хранится по пути:

```text
Документы\BuoyCalc\Libraries\user-buoys.json
```

## Архитектурное решение

Главное окно теперь постепенно становится окном проекта:

```text
Главное окно проекта
├─ выбрать буй из библиотеки
├─ собрать последовательность элементов
├─ задать условия
├─ выбрать / задать якорь
└─ рассчитать / сформировать отчёт
```

Отдельное окно библиотеки должно постепенно стать общим редактором элементов:

```text
Библиотека элементов
├─ буи
├─ линии
├─ соединители
├─ приборы
└─ якоря
```

## Как обновиться локально

Если репозиторий уже клонирован, в Visual Studio можно использовать:

```text
Git → Pull
```

Затем запустить проект заново.

## Как проверить библиотеку элементов

1. В верхней панели главного окна нажать `Библиотека элементов...`.
2. В новом окне выбрать встроенный буй.
3. Навести мышкой на `?` рядом с параметрами буя.
4. Нажать `Новый буй`.
5. Заполнить параметры.
6. Нажать `Сохранить`.
7. Проверить, что новый буй появился в списке.
8. Закрыть окно библиотеки.
9. В главном окне проверить, что список буёв обновился.
10. Выбрать новый буй и нажать `Рассчитать систему`.

## Как проверить сохранение проекта

1. Изменить название проекта.
2. Выбрать или создать буй.
3. Добавить или удалить элемент последовательности.
4. Нажать `Сохранить как...` и выбрать папку/имя файла.
5. Нажать `Новый`.
6. Нажать `Загрузить...` и выбрать сохранённый файл.
7. Проверить, что проект восстановился.

## Как запустить на Windows

### 1. Установить .NET SDK

Установить .NET SDK 8 или новее.

### 2. Открыть папку проекта

Открыть папку репозитория `buoycalc-windows` в Visual Studio, Rider или VS Code.

### 3. Восстановить пакеты

В терминале внутри папки проекта:

```bash
dotnet restore
```

### 4. Запустить

```bash
dotnet run
```
