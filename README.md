# BuoyCalc Windows v0.11.2

Windows-ветка проекта BuoyCalc на **C# + Avalonia**.

Это портирование расчётной логики и интерфейса в desktop-приложение:

```text
SwiftUI iPhone prototype
↓
C# / Avalonia Windows desktop prototype
```

## Статус

v0.11.2 исправляет ошибку запуска после v0.11.1: внешний `DataGrid` убран, потому что сборка `Avalonia.Controls.DataGrid` не подтянулась в проекте. Таблица элементов теперь сделана на штатных элементах Avalonia и должна собираться без дополнительных NuGet-пакетов.

## Что изменилось в v0.11.2

1. удалён пакет `Avalonia.Controls.DataGrid`;
2. удалён `StyleInclude` для `DataGrid` из `App.axaml`;
3. таблица элементов заменена на стабильную встроенную таблицу;
4. таблица имеет горизонтальную и вертикальную прокрутку;
5. Markdown-отчёт оставлен ниже для копирования;
6. в отчёте длинные строки не переносятся, а прокручиваются горизонтально.

## Как проверить v0.11.2

1. Запустить программу.
2. Нажать `Рассчитать систему`.
3. В правой колонке найти блок `Таблица элементов`.
4. Проверить, что сборка проходит без ошибок `DataGrid`.
5. Проверить, что таблица читается и не разваливается на переносах строк.
6. Проверить горизонтальную прокрутку таблицы.
7. Проверить блок `Отчёт`: длинные строки должны прокручиваться горизонтально.

## Что важно понимать

v0.11.2 не меняет физику расчёта, а исправляет сборку и возвращает читаемую таблицу. Перетаскивание ширины колонок будет сделано позже отдельным стабильным механизмом без внешнего `DataGrid`.

Текущие упрощения:

1. линия считается по проецируемой площади `длина × диаметр`;
2. волновая составляющая пока прикладывается только к бую;
3. якорь пока считается через вес в воде, коэффициент удержания и грунтовый множитель;
4. динамика, усталость, рывки, провисание линии и контакт с грунтом пока не моделируются.
